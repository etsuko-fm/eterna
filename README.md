# Eterna


Eterna is a 6-voice sampler and sequencer for creating rhythmic textures and patterns from a single audio file.

## Requirements

- monome norns
- A WAV/AIFF sample (48kHz), which may be mono, stereo or up to 6 channels.

Eterna is designed to have a fast and enjoyable workflow for users with just a norns. Additionally, the following hardware is supported:
- monome grid
- midi controller
